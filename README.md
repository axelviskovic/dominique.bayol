## Le projet

Tout le projet doit être compilé avec Gulp. Pour cela :

- Installez Gulp sur votre système
- Run ```npm install```
- Run ```gulp watch``` et CTRL+C --> *localhost:5000* CTRL+V dans votre navigateur
